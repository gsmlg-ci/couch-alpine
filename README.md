# couch-alpine

[![Build](https://github.com/gsmlg-ci/couch-alpine/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/couch-alpine/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/couch-alpine`
- `ghcr.io/gsmlg-dev/couch-alpine`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/couch-alpine:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/couch-alpine:latest
```

## Build

```bash
docker build -t gsmlg/couch-alpine .
```
